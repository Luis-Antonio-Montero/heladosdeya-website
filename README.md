# 🍦 Helados de Ya — Sitio Web Oficial

Sitio web desarrollado para **Suplidora y Heladería Deya**, negocio local de helados artesanales en República Dominicana.

## 🔗 Demo en vivo

[Ver sitio →](https://heladosdeya.shop)

## 🛠️ Stack tecnológico

### Frontend
- HTML5 / CSS3 (estructura y estilos personalizados)
- JavaScript Vanilla (ES6+)
- Bootstrap 5.3.3 (grid, componentes UI)
- Bootstrap Icons 1.11.3
- Tailwind CSS (utilidades de estilo)
- Google Fonts — Mulish

### Backend
- PHP (API REST propia)
- Endpoints: catálogo de productos, carrito, configuración

### Integraciones
- WhatsApp API (envío de pedidos por mensaje directo)
- localStorage (persistencia del carrito sin login)

## 📋 Características

- Diseño responsive para móvil y desktop
- Catálogo de 40+ productos dividido en categorías (Conos, Paletas, Cubos, Suministro, Tarros)
- Filtro de productos por categoría y búsqueda por nombre
- Carrito de compras persistente (localStorage)
- Envío de pedido directo por WhatsApp con mensaje pre-formateado
- Venta al por mayor y al detalle
- Sin registro de usuario requerido

## 🗂️ Estructura del proyecto

```
heladosdeya.shop/
├── index.html          # Página principal
├── productos.html      # Catálogo completo
├── estilos.css         # Estilos personalizados
├── script.js           # Lógica del carrito y catálogo
├── img/                # Imágenes de productos y logo
└── api/
    ├── health.php          # Estado de la API
    └── public/
        ├── bootstrap.php   # Configuración inicial
        └── cart.php        # Gestión del carrito
```
